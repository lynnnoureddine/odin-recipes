A beginner-friendly HTML project showcasing **three classic dessert recipes**:

- Cinnamon Rolls
- Creamy Banana Bread
- Apple Pie

Each recipe includes:

- A list of ingredients
- Step-by-step instructions
- Tips & tricks for best results
- Optional toppings or serving suggestions

This project was completed and submitted as part of **The Odin Project**, demonstrating my understanding and practice of **HTML basics**, including:

- Semantic HTML elements (headings, paragraphs, lists)
- Structuring content with sections
- Incorporating images
- Creating organized and readable markup
